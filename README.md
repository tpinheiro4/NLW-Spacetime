## Projeto
O projeto é uma capsula do tempo para mostrar memórias em uma linha do tempo.

## Tecnologias
Esse projeto foi desenvolvido durante a NLW da Rockeseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Git Hub
